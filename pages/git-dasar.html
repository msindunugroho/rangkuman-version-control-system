Git

  1. Glosarry

    1. Repository

      Tempat penyimpanan untuk proyek perangkat lunak yang menggunakan Git untuk kontrol versi.

    2. Commit

      Tindakan menyimpan perubahan pada file atau proyek ke repositori Git.

    3. Branch

      Cabang dari repositori yang memungkinkan pengembang untuk bekerja secara terpisah dari kode yang ada.

    4. Merge

      Penggabungan perubahan dari satu branch ke branch lainnya.

    5. Pull Request

      Permintaan untuk menggabungkan perubahan dari satu branch ke branch lainnya, biasanya digunakan dalam kolaborasi tim.

    6. Push

      Mengirimkan perubahan lokal ke repositori Git yang terkait.

    7. Pull

      Mengambil perubahan dari repositori Git yang terkait dan menggabungkannya dengan perubahan lokal.

    8. Fetch

      Mengambil semua perubahan dari repositori Git yang terkait, tetapi tidak melakukan penggabungan otomatis dengan perubahan lokal.

    9. Clone

      Membuat salinan dari repositori Git yang ada di server ke komputer lokal.

    10. Remote

      Referensi ke repositori Git yang berada di tempat lain, biasanya digunakan untuk berinteraksi dengan repositori di server.

    11. Origin

      Nama default yang diberikan ke remote repository ketika melakukan clone.

    12. Revert

      Membatalkan perubahan tertentu pada commit tertentu.

    13. Fork

      Membuat salinan repositori milik orang lain di GitHub ke akun GitHub Anda sendiri.
